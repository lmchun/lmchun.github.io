Mind Loft

HTC Vive, Unity, 2019

Lauren Chun: 3D Animator, Modeler, Texturing, Level Design, Designer Kevin Ta: Programming, Writing, Voice Acting, Producer

This is a project that I worked on with an interdisciplinary team to create an educational VR game that teaches players about what parts of the brain are used to do specific functions.

The name, Mind Loft refers to the memory palace that is a term used in psychology for memorizing things. When you have to remember something, we picture these objects in room in a familar place, in this case a loft. This is a prototype that was made to see what the potential of using VR to help solidify learning in a way that cannot be shown or demonstrated in reality or classic 2D/3D computer games.

This project went through many iterations and started with a whole different concept but still trying to use an experimental medium to convey learning about the mind and concepts of it.
